To get started with Scho locally:

### 1. Clone the Repository

```bash
git clone https://github.com/ahmadrafidev/scho.git
cd scho
```

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Run the Dev Server

```bash
pnpm dev
```
